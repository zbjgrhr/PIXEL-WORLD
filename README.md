# Pixel World — Prompt to Play

输入一个游戏构想，系统会先把自然语言整理为结构化游戏规格，再分别生成互不污染的像素素材，并组装成可玩的 2D 横版跳跃战斗游戏。

## 这次升级包含什么

- 一键提示词优化：将自由描述拆分为世界观、主角、敌人、武器、关卡、Boss 等明确字段。
- 素材隔离编译：背景、主角、敌人、武器、地面、障碍物、弹射物、攻击特效、收集品和 Boss 使用各自独立的提示词与反向约束。
- AI 与本地双重优化：有可用 API Key 时调用文本模型；不可用时自动使用本地结构化编译器，不会阻断流程。
- 完整战斗：近战、远程弹射物、普通敌人追踪/巡逻/射击、伤害、生命值、死亡和无敌帧。
- 完整关卡：多关卡、收集品、障碍物碰撞、击败全部敌人后解锁出口、最终 Boss 和胜利结算。
- 统一视觉合同：所有素材共享像素规格、色板、视角和光照，但不会共享不属于自己的内容。
- 独立预览与重生成：每类素材都可单独检查、抠图和重新生成。
- 兼容 DashScope、OpenAI 和 OpenRouter 图像模型。

## 运行

```bash
pnpm install
pnpm dev
```

打开 `http://localhost:3000`。

你可以在页面中临时输入 API Key，或使用环境变量：

```env
DASHSCOPE_API_KEY=your-key-here
OPENAI_API_KEY=your-key-here
OPENROUTER_API_KEY=your-key-here
```

API Key 不应写入源码、README、PRD 或 Git 提交记录。

## 使用流程

1. 选择图像服务商和模型。
2. 输入主题名称与游戏构想。
3. 点击“一键优化提示词”，检查结构化结果。
4. 选择关卡数量并创建主题。
5. 在预览区检查十类素材，需要时单独重新生成。
6. 点击 Start Game 开始游戏。

## 游戏操作

- `A/D` 或 `←/→`：移动
- `W` 或 `Space`：跳跃
- `J`：近战攻击
- `K` 或 `F`：远程攻击
- `Esc`：暂停/继续

移动端提供屏幕按钮。每关必须击败全部普通敌人或 Boss，出口才会解锁。

## 核心结构

- `app/api/optimize-prompt`：文本提示词优化与结构化规格生成。
- `lib/game-spec.ts`：规格默认值、校验、迁移与序列化。
- `lib/game-prompts.ts`：按素材类别编译严格隔离的正向/反向提示词。
- `app/api/generate`：生成全局战斗素材、逐关场景素材和生成关卡配置。
- `components/GameCanvas.tsx`：平台移动、战斗、敌人 AI、弹射物、收集品、Boss 与关卡推进。
